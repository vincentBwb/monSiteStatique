Input Types

MDN Web Docs : input tag

Button

MDN Web Docs : input type "button"

Un bouton sans comportement défini


Checkbox

MDN Web Docs : input type "checkbox"

Une case à cocher qui permet de sélectionner/déselectionner une valeur


Color

MDN Web Docs : input type "color"

HTML5 un contrôle qui permet de définir une couleur


Date

MDN Web Docs : input type "date"

HTML5 un contrôle qui permet de saisir une date (composé d'un jour, d'un mois et d'une année)


Datetime-local

MDNMDN Web Docs : input type "datetime-local"

HTML5 un contrôle qui permet de saisir une date et une heure (sans fuseau horaire)


Email

MDN Web Docs : input type "email"

HTML5 un champ qui permet de saisir une adresse éléctronique


File

MDN Web Docs : input type "file"

Un contrôle qui permet de sélectionner un fichier
L'attribut accept définit les types de fichiers qui peuvent être sélectionnés


Hidden

MDN Web Docs : input type "hidden"

Un contrôle qui n'est pas affiché mais dont la valeur est envoyée au serveur


Image

MDN Web Docs : input type "image"

Un bouton graphique d'envoi du formulaire
L'attribut src doit être défini avec la source de l'image et l'attribut alt doit être défini avec le texte alternatif
Les attributs height et width permettent de définir la taille de l'image en pixels


Month

MDN Web Docs : input type "month"

HTML5 un contrôle qui permet de saisir un mois et une année (sans fuseau horaire)


Number

MDN Web Docs : input type "number"

HTML5 un contrôle qui permet de saisir un nombre


Password

MDN Web Docs : input type "password"

Un champ texte sur une seule ligne dont la valeur est masquée
Les attributs maxlength et minlength définissent la taille maximale et minimale de la valeur à saisir dans le champ


Radio

MDN Web Docs : input type "radio"

Un bouton radio qui permet de sélectionner une seule valeur parmi un groupe de différentes valeurs


Range

MDN Web Docs : input type "range"

HTML5 un contrôle qui permet de saisir un nombre dont la valeur exacte n'est pas importante


Reset

MDN Web Docs : input type "reset"

Un bouton qui réinitialise le contenu du formulaire avec les valeurs par défaut


Search

MDN Web Docs : input type "search"

HTML5 un champ texte sur une ligne pour des termes de recherche. Les sauts de ligne sont automatiquement retirés


Submit

MDN Web Docs : input type "submit"

Un bouton qui envoie le formulaire


Tel

MDN Web Docs : input type "tel"

HTML5 un contrôle pour saisir un numéro de téléphone


>

Text

MDN Web Docs : input type "text"

Un champ texte sur une seule ligne. Les sauts de ligne sont automatiquement retirés


Time

MDN Web Docs : input type "time"

HTML5 Un contrôle pour entrer une valeur de temps (sans fuseau horaire)


Url

MDN Web Docs : input type "url"

HTML5 un champ permettant de saisir une URL


Week

MDN Web Docs : input type "week"

HTML5 un contrôle permettant de saisir une date représentée par un numéro de semaine et une année (sans fuseau horaire)



Others Tags


Select tag

MDN Web Docs : select tag

Un contrôle qui permet de sélectionner une seule valeur parmi une liste de différentes valeurs


Datalist tag

MDN Web Docs : datalist tag

Un champ texte qui propose l'auto-complétion parmi une liste de différentes valeurs


Textarea tag

MDN Web Docs : textarea tag

Un contrôle qui permet d'éditer du texte sur plusieurs lignes


Button tag

MDN Web Docs : button tag

Un contrôle interactif ayant la forme d'un bouton et qui pourra être utilisé dans un formulaire ou dans le document


Img tag

MDN Web Docs : img tag

Un contrôle qui permet d'afficher une image dans un document


Audio tag

MDN Web Docs : audio tag

Un contrôle qui permet d'intégrer un contenu sonore dans un document


Video tag

MDN Web Docs : video tag

Un contrôle qui permet d'intégrer une video dans un document


Canvas tag

MDN Web Docs : canvas tag

Un contrôle qui permet de modifier une zone graphique via un script (habituellement en JavaScript ou grâce à WebGL)
Il peut par exemple être utilisé afin de dessiner des graphiques, manipuler des images ou jouer des animations
