Le mode strict

  • Intérêt du mode strict
    • Il est un langage plus strict que javascript. Il relève des erreurs que js ne relevera dans un grand script. Il ne s'executera pas
    • exemples :
      • Une variable déclaré sans le mot clé var
      • Une valeur assignée à une variable ou une propriété qui est en lecture seule
      • Supprimer une variable ou une propriété qui n' est pas suprimable
      • Définir une variable ou une propriété plus d'une fois
      • Dupliquer le nom d'un paramètre
      • ou quand on tente d'utiliser une syntaxe octale ou des caractères d'échappements
    • La méthode eval n'est pas authoriser à créer des variables depassant sa porpre portée
  • Utilisation
    • De manière globale ou locale. On utilise l'expression use strict
